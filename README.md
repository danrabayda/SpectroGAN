# SpectroGAN 

A generative adversarial network (GAN) for generating waveforms using spectrograms for training.

How to run:
 - git clone this repo
 - in SpectroGAN folder run "pip install ."
 - open [notebooks/train.ipynb](https://github.com/danrabayda/SpectroGAN/blob/main/notebooks/train.ipynb) and run it

\
\
Training data sources:
 - ESC50:
https://www.kaggle.com/datasets/mmoreaux/environmental-sound-classification-50?resource=download
